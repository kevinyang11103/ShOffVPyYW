# 项目简介 python984

本项目是一个基于Python深度学习的音乐推荐系统，采用B/S架构，以Django框架为后端，MySQL数据库存储数据。以下是项目的详细介绍。

## 技术栈

- **后端语言**: Python
- **框架**: Django
- **数据库**: MySQL
- **推荐算法**: 基于深度学习的KNN数据挖掘和分析算法

## 功能模块

- 用户管理
  - 用户注册与登录
  - 用户音乐喜好设置
- 音乐推荐
  - 基于用户历史听歌数据的个性化推荐
  - 热门音乐推荐
- 数据分析
  - 用户听歌行为分析
  - 音乐流行趋势分析

## 系统角色

- **普通用户**: 可以浏览、搜索音乐，设置个人喜好，获取个性化推荐
- **管理员**: 管理用户信息，监控系统运行状态，处理异常

## 运行环境

- **操作系统**: 兼容Linux和Windows
- **Python版本**: 3.6 或以上
- **依赖库**: Django, NumPy, Pandas, scikit-learn, TensorFlow

## 部署步骤

1. 环境配置：安装Python及依赖库
2. 数据库部署：配置MySQL数据库
3. 后端部署：运行Django项目
4. 前端部署：根据后端API部署相应的前端页面

## 核心亮点

- **个性化推荐**: 采用深度学习算法，为用户提供更加精准的音乐推荐
- **数据分析**: 深入分析用户听歌行为和音乐流行趋势
- **易于扩展**: 基于Django框架，方便后续功能模块的添加和扩展
